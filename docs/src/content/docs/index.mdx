TCGer is a multi-game trading card collection manager with a Node/Express API, a Next.js frontend, and optional local cache services.

Supported games:

- Yu-Gi-Oh!
- Magic: The Gathering
- Pokemon

## What you can do here

- Set up the stack with Docker.
- Run backend and frontend locally.
- Explore API docs and key endpoints.
- Understand repo structure and architecture decisions.

## Start here

1. Follow [Getting Started](/getting-started/) for Docker setup.
2. Use [Local Development](/local-development/) if you want to run services manually.
3. Read the [API Guide](/reference/api/) for endpoint and OpenAPI details.
